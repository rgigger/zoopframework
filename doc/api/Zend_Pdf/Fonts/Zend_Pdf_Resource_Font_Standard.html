Class Zend_Pdf_Resource_Font_Standard

Description
Description | Descendents | Vars | Methods (details)

Abstract class definition for the standard 14 Type 1 PDF fonts.

The standard 14 PDF fonts are guaranteed to be availble in any PDF viewer implementation. As such, they do not require much data for the font's resource dictionary. The majority of the data provided by subclasses is for the benefit of our own layout code.

The standard fonts and the corresponding subclasses that manage them:

  • Courier - Zend_Pdf_Resource_Font_Standard_Courier
  • Courier-Bold - Zend_Pdf_Resource_Font_Standard_CourierBold
  • Courier-Oblique - Zend_Pdf_Resource_Font_Standard_CourierOblique
  • Courier-BoldOblique - Zend_Pdf_Resource_Font_Standard_CourierBoldOblique
  • Helvetica - Zend_Pdf_Resource_Font_Standard_Helvetica
  • Helvetica-Bold - Zend_Pdf_Resource_Font_Standard_HelveticaBold
  • Helvetica-Oblique - Zend_Pdf_Resource_Font_Standard_HelveticaOblique
  • Helvetica-BoldOblique - Zend_Pdf_Resource_Font_Standard_HelveticaBoldOblique
  • Symbol - Zend_Pdf_Resource_Font_Standard_Symbol
  • Times - Zend_Pdf_Resource_Font_Standard_Times
  • Times-Bold - Zend_Pdf_Resource_Font_Standard_TimesBold
  • Times-Italic - Zend_Pdf_Resource_Font_Standard_TimesItalic
  • Times-BoldItalic - Zend_Pdf_Resource_Font_Standard_TimesBoldItalic
  • ZapfDingbats - Zend_Pdf_Resource_Font_Standard_ZapfDingbats

Font objects should be normally be obtained from the factory methods Zend_Pdf_Font::fontWithName and Zend_Pdf_Font::fontWithPath.
